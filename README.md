## Project Setup

## Laravel
```sh
compose install
```
```sh
php artisan serve
```
## Vue js

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
